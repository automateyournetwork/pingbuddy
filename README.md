# pingbuddy
AI Analysis and "chat" with your PING

## Getting started

Clone the repo

Make a .env file inside the packet_buddy folder (where the ping_buddy.py file is located)

put this in the file:
```console
OPENAI_API_KEY="<your openapi api key>"
```

## Bring up the server
docker-compose up 

## Visit localhost
http://localhost:8506
